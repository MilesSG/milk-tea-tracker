# 奶茶日记 (Milk Tea Diary)

一个帮助你追踪每日奶茶消费的微信小程序。

## 功能特点

- 📝 记录每日奶茶消费
- 📊 查看消费统计和趋势图
- 🎯 设置每月奶茶目标
- 💖 精美的粉色系界面设计
- 🏷️ 支持多个奶茶品牌选择

## 技术栈

- 微信小程序原生开发
- Canvas 绘图
- 本地数据存储

1. 克隆仓库

```bash

```

2. 使用微信开发者工具打开项目

3. 编译运行即可
